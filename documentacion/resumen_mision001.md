# Misión 001 - Laboratorio inicial con VS Code y python

## Scripts realizados

### escaneo_ping.py
- Escanea IPs del 192.168.1.1 al 192.168.1.20

### analizador_logs.py
- Lee logs línea por línea
- Busca palabras como "failed", "ssh", "error"
- Ideal para análisis forense o troubleshooting

- Terminal integrada
